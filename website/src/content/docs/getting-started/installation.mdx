InstantDB Flutter is a real-time, offline-first database client that provides reactive bindings for Flutter applications. It enables you to build collaborative, real-time applications with ease.

## Requirements

- Flutter SDK >= 3.8.0
- Dart SDK >= 3.8.0
- An InstantDB app ID (get one at [instantdb.com](https://instantdb.com))

## Installation

Add InstantDB Flutter to your `pubspec.yaml`:

```yaml
dependencies:
  instantdb_flutter: ^0.1.0
```

Or install from the command line:

<Tabs>
  <TabItem label="Flutter">
    <Code code="flutter pub add instantdb_flutter" lang="sh" />
  </TabItem>
  <TabItem label="Dart">
    <Code code="dart pub add instantdb_flutter" lang="sh" />
  </TabItem>
</Tabs>

## Import

Add the import to your Dart files:

```dart
import 'package:instantdb_flutter/instantdb_flutter.dart';
```

## Get Your App ID

1. Visit [instantdb.com](https://instantdb.com) and create an account
2. Create a new app
3. Copy your App ID from the dashboard
4. **Store your App ID in a `.env` file** in your project root:
   ```env
   INSTANTDB_APP_ID=your-app-id-here
   ```
5. Use it in your Flutter app initialization
